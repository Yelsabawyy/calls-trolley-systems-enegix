# Calls Trolley Systems (Enegix)

Vite + React + TypeScript + Tailwind + Dexie + PWA project.

## Setup
```bash
git clone https://github.com/Yelsabawyy/calls-trolley-systems-enegix.git
cd calls-trolley-systems-enegix
npm install
```

## Run Locally
```bash
npm run dev
```

Open http://localhost:5173
